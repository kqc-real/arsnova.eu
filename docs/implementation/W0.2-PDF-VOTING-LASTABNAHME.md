# W0.2 – PDF-vs.-Live-Voting-Lastabnahme

**Stand:** 2026-07-24  
**Szenario:** `pdf-vs-live-voting-500`  
**Reproduzierbarer Runner:** `scripts/load/pdf-vs-live-voting-500.mjs`

## Ziel

Die W0.2-Akzeptanzbedingung aus dem Security-Hardening-Plan wird unter echter
tRPC-/PostgreSQL-/Redis-/Playwright-Last geprüft:

- 500 Clients von derselben Quell-IP mit 500 unterschiedlichen Participant-IDs
- repräsentative PDF-Testsession mit 500 Teilnehmenden und 10.000 Votes über 20 Fragen
- separater 500-Vote-Baseline-Lauf ohne PDF-Last
- zwei gleichzeitig laufende Playwright-PDF-Jobs
- sofortige 429-Ablehnung eines weiteren PDF-Requests
- Vote-p95 unter 1.500 ms, Vote-p99 unter 3.000 ms und Fehlerquote unter 1 %
- sichtbare PDF-Auslastungs- und Ablehnungsmetriken
- Rückkehr auf null aktive PDF-Jobs und erfolgreicher PDF-Export ohne Neustart

## Umgebung und Befehl

Der erste repräsentative Lauf erfolgte lokal auf macOS gegen den echten Backend-Prozess des PR-Branches (12 logische CPUs, 16 GiB RAM, arm64, Node 20.19.0).
PostgreSQL und Redis liefen in den Projekt-Containern; alle 500 Clients verwendeten über
Loopback dieselbe Quell-IP `127.0.0.1`. Es wurden keine Resolver, Datenbankzugriffe,
Playwright-Aufrufe oder Telemetriepfade gemockt.

```bash
REPORT_FILE=output/load/pdf-vs-live-voting-500.json \
  npm run load:pdf-vs-voting:500
```

Der Runner erzeugt eine beendete Session mit 500 Teilnehmenden und 10.000 Votes über 20
Fragen für die konkurrierenden Playwright-Berichte. Anschließend misst er einen separaten
500-Vote-Baseline-Burst ohne PDF-Last und einen zweiten 500-Vote-Burst unter zwei dauerhaft
aktiven PDF-Jobs.

## Einordnung

Der lokale Lauf belegt die Protokoll-, Parallelitäts-, Metrik- und Recovery-Eigenschaften
des W0.2-Slices mit einem repräsentativen großen Ergebnisgraphen und separater Baseline. Er
ersetzt noch keine Kapazitätsaussage für den Linux-Zielhost. Für die endgültige W0.2-Abnahme
wird derselbe Runner deshalb zusätzlich in einem isolierten Produktionscontainer auf dem
Zielhost ausgeführt; dessen Hardware- und Rohmetriken werden in diesem Bericht ergänzt.
